Stella
http://www.iseesystems.com/
El software Stella permite la creación de modelos computacionales desde una interfaz gráfica, esta presenta cuatro niveles de trabajo, que se describen a continuación utilizando el modelo Pehuen de ejemplo.
El primer nivel del programa es la interfaz del modelo que se crea, no es necesario generarla pero si es muy útil luego de haber producido el modelo de simulación, esto es necesario cuando se desea que usuarios no familiarizados con el modelo puedan operar con él. En este caso se observa los módulos que componen el modelo y los distintos enlaces entre ellos.
En este caso también se muestra una parte de la interfaz que permite configurar valores iniciales de algunas variables del modelo y las relaciones más importantes, como la carga ganadera, la tasa de natalidad , de mortalidad en relación al módulo ganadero, pero también se pueden configurar los otros módulos. Algunas relaciones que se pueden modificar valores son la precipitación anual, el esfuerzo de recolección de semillas, entre otros.
El otro nivel importante en el programa Stella es el del mapa o modelo propiamente dicho, en él y a través de los distintos elementos gráficos se construye el modelo, y se establece la relación entre las distintas variables, cuáles son los stocks, los flujos y otros valores relevantes, también se asignan lo valores de constantes. Los distintos componentes pueden ser asignados a los módulos, que se utilizan para organizar el sistema.
El último nivel es el de las ecuaciones, estás se generan de forma automática al crear el diagrama con los elementos gráficos, por lo general no se emplea más que para tener una referencia de como el sistema está constituido.
